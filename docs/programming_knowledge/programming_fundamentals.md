### 1. Java
- **Foco:** Desenvolvimento backend, sistemas empresariais, arquitetura de microsserviços.
- **Conceitos Chave:** Programação Orientada a Objetos (POO), JVM, Spring Framework (Boot, Data, Security), Maven/Gradle, testes unitários (JUnit, Mockito).

### 2. TypeScript
- **Foco:** Desenvolvimento frontend e backend (Node.js), tipagem estática para JavaScript.

### 3. React

- **Conceitos Chave:** Componentes (funcionais e de classe), Hooks, gerenciamento de estado (Context API, Redux/Zustand), roteamento (React Router), testes (Jest, React Testing Library).

### 4. HTML e CSS

- **Conceitos Chave:** Semântica HTML5, CSS3 (Flexbox, Grid), pré-processadores (Sass/Less), metodologias CSS (BEM, ITCSS), responsividade (Media Queries).

### 5. Python
- **Foco:** Scripting, automação, análise de dados, inteligência artificial, desenvolvimento backend (Django, Flask).
- **Conceitos Chave:** Estruturas de dados, algoritmos, bibliotecas (NumPy, Pandas, Scikit-learn, TensorFlow/PyTorch), desenvolvimento web (Flask/Django), automação de tarefas.

### 6. Luau (para Roblox)
- **Foco:** Desenvolvimento de jogos e experiências na plataforma Roblox.
- **Conceitos Chave:** Sintaxe Luau (similar ao Lua), Roblox API, programação orientada a eventos, otimização de performance para jogos, criação de sistemas de jogo (inventário, combate, UI).
