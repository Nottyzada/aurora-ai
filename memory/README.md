# Memória Persistente da Aurora

Este diretório é destinado ao armazenamento da memória persistente da Aurora, gerenciada pelo Hermes Agent. Aqui serão guardados:

- **Experiências e Interações:** Registros de conversas e tarefas executadas.
- **Habilidades Aprendidas:** Novas habilidades e automações criadas pela Aurora.
- **Contexto de Longo Prazo:** Informações relevantes para o aprendizado contínuo e a personalização do comportamento da IA.

O Hermes Agent utiliza este diretório para manter o estado da Aurora entre as sessões, garantindo que ela "lembre" de interações passadas e continue a evoluir.
